## 1. 产品简介

MindBridge Assist 是一款面向心智障碍群体的数字化辅助支持系统，帮助用户进行心理评估、行为能力分析、情绪追踪、沟通辅助及需求表达，并为家属、教师、治疗师和医疗机构提供协作平台。

**核心能力：**

- 🧠 标准化心理评估与 AI 辅助分析
- 📊 行为能力追踪与模式识别
- 😊 情绪识别与趋势管理
- 💬 AAC 增强替代沟通工具
- 🤝 多角色协作与信息共享
- 📈 智能报告生成与数据可视化

## 2. 快速开始

### 2.1 注册账号

1. 访问 MindBridge Assist 官网或打开 App
2. 点击 **「注册」** 按钮
3. 填写以下信息：
   - 手机号码或电子邮箱

   - 选择您的角色（家属 / 教师 / 治疗师 / 医生 / 机构管理员）
4. 阅读并同意《用户服务协议》与《隐私政策》

6. 验证手机短信或邮箱验证码

## 4. 功能操作指南

### 4.1 心理评估流程

#### 4.1.1 选择评估量表

1. 进入首页，点击 **「心理评估」** 模块
2. 选择需要评估的用户（如管理多个用户）
3. 浏览可用的评估量表，可按类别筛选：
   - 认知能力评估
   - 情绪健康评估
   - 社交能力评估
   - 适应行为评估
   - 综合发展评估
4. 点击量表名称查看详情（适用年龄、评估时长、量表说明）
5. 点击 **「开始评估」**

> 📸 *[截图位置：评估量表列表页]*

#### 4.1.2 执行评估

1. 评估页面会显示当前题目与进度条
2. **自助评估：** 用户根据提示作答（系统会自动适配呈现方式）
3. **代填评估：** 由教师/治疗师/家属根据观察代为填写
4. 每题作答后自动保存，可点击 **「上一题」** 修改
5. 全部完成后点击 **「提交评估」**

> 💡 **AI 辅助模式：** 系统会根据用户之前的作答情况，自动调整后续题目的呈现难度与方式。

> 📸 *[截图位置：评估执行页面]*

#### 4.1.3 查看评估结果

1. 提交后约 10-30 秒自动生成评估结果
2. 结果页面包含：
   - 各维度得分与等级
   - 与常模的对比图表
   - AI 生成的解读文字
   - 建议的后续行动
3. 可点击 **「分享结果」** 发送给协作团队成员
4. 可点击 **「导出 PDF」** 保存报告

> 📸 *[截图位置：评估结果页面]*

#### 4.1.4 评估结果解读指南

评估结果生成后，正确理解各指标含义对于制定后续干预方案至关重要。以下是结果页面中各关键要素的解读说明：

**各维度得分含义：**

| 维度       | 得分范围   | 含义说明                          |
| -------- | ------ | ----------------------------- |
| 认知能力     | 0-100  | 反映信息处理、逻辑推理与问题解决的综合能力        |
| 情绪健康     | 0-100  | 反映情绪稳定性、情绪调节能力与主观幸福感         |
| 社交能力     | 0-100  | 反映社会互动、沟通意愿与人际关系建立能力         |
| 适应行为     | 0-100  | 反映日常生活自理、社区适应与功能独立性          |
| 综合发展     | 0-100  | 以上各维度的加权综合得分                 |

**百分位含义：**

- 百分位表示该用户得分在同龄同性别常模群体中的相对位置
- 例如百分位为 75，表示该用户得分高于常模群体中 75% 的人
- 百分位仅反映相对位置，不直接等同于能力高低，需结合具体维度综合判断

**临床临界值说明：**

| 临界区间      | 百分位范围    | 临床意义                     | 建议操作         |
| --------- | ------- | ------------------------ | ------------ |
| 正常范围      | ≥ 25    | 发展水平在正常范围内               | 继续常规监测       |
| 边缘范围      | 16-24   | 处于正常与需关注之间的边界地带          | 建议增加监测频率     |
| 需关注范围     | 5-15    | 低于大多数同龄人，可能需要干预支持        | 建议咨询专业治疗师    |
| 需重点干预范围   | < 5     | 显著低于同龄人水平，需尽快启动专业干预      | 尽快联系专业机构评估   |

> ⚠️ **重要提示：** 临床临界值仅供参考，不构成医学诊断。最终诊断需由持证专业人员结合多方面信息综合判断。

**AI 解读的参考范围：**

- AI 生成的解读文字基于常模数据与临床研究文献，提供参考性分析
- 解读中标注的"参考范围"指同龄同性别群体的平均表现区间
- AI 解读末尾会标注置信度等级（高/中/低），置信度越高，解读越可靠
- 如 AI 解读与专业判断不一致，请以专业人员的临床判断为准

> 📸 *[截图位置：评估结果解读页面]*

#### 4.1.5 评估历史与对比

1. 进入 **「心理评估」** → **「历史记录」**
2. 查看该用户所有已完成的评估列表，按时间倒序排列
3. 点击任意评估可查看详细结果
4. **多次评估对比：**
   - 勾选 2-5 次历史评估，点击 **「对比分析」**
   - 系统将生成对比视图，包含：
     - 各维度得分变化雷达图
     - 各维度得分趋势折线图
     - 百分位变化对照表
5. **进步/退步标识：**
   - 📈 **进步：** 维度得分较上次提升 ≥ 5 分，以绿色箭头标识
   - ➡️ **持平：** 维度得分变化 < 5 分，以灰色横线标识
   - 📉 **退步：** 维度得分较上次下降 ≥ 5 分，以红色箭头标识
6. 可导出对比报告（PDF 格式）

> 💡 **提示：** 建议每次干预周期结束后（通常 3-6 个月）进行一次复评，以便追踪干预效果。

> 📸 *[截图位置：评估历史与对比页面]*

### 4.2 行为记录与分析

#### 4.2.1 记录行为事件

1. 进入 **「行为分析」** 模块
2. 点击 **「+ 新增记录」** 按钮
3. 填写行为事件信息：

| 字段        | 说明                     |
| --------- | ---------------------- |
| **用户**    | 选择记录对象                 |
| **日期与时间** | 自动填充当前时间，可修改           |
| **前因（A）** | 行为发生前的环境/事件（支持文字、语音输入） |
| **行为（B）** | 具体描述行为表现（可快速选择预设标签）    |
| **结果（C）** | 行为发生后的结果或反应            |
| **严重程度**  | 1-5 级滑动选择              |
| **备注**    | 补充说明（可选）               |

4. 点击 **「保存」**

> 💡 **快捷记录：** 在 AAC 沟通工具页面长按任意按钮，可快捷跳转至行为记录。

> 📸 *[截图位置：行为记录表单]*

#### 4.2.2 查看行为分析

1. 进入 **「行为分析」** → **「分析视图」**
2. 选择时间范围（近 7 天 / 近 30 天 / 自定义）
3. 查看以下内容：
   - **行为频率趋势图：** 各类行为事件的数量变化
   - **行为功能分析：** AI 识别的行为模式与功能假设
   - **ABC 关联图：** 前因-行为-结果的关联可视化
   - **能力发展曲线：** 目标行为能力随时间的变化

> 📸 *[截图位置：行为分析仪表盘]*

#### 4.2.3 设置行为目标

1. 进入 **「行为分析」** → **「目标管理」**
2. 点击 **「+ 新增目标」**
3. 设定目标信息：
   - 目标名称（如「减少课堂中的离座行为」）
   - 目标行为描述
   - 基线值（当前水平）
   - 目标值（期望达到的水平）
   - 目标期限
4. 系统会自动追踪目标达成进度并推送提醒

> 📸 *[截图位置：行为目标设置页面]*

### 4.3 情绪日志与趋势查看

#### 4.3.1 记录情绪

**自助记录（适合有一定表达能力的用户）：**

1. 打开 App，首页即显示 **「今天心情如何？」**
2. 从可视化情绪图标中选择当前感受：
   - 😄 很开心 → 😊 不错 → 😐 一般 → 😟 有点不好 → 😢 很难过
3. 可选补充信息：
   - 选择情绪原因（预设标签：学习、社交、家庭、身体等）
   - 语音补充说明（可选）
4. 点击 **「记录」**

**代填记录（家属/教师/治疗师代为记录）：**

1. 进入 **「情绪追踪」** → **「记录情绪」**
2. 选择用户
3. 根据观察选择情绪状态
4. 补充观察备注
5. 点击 **「保存」**

> 📸 *[截图位置：情绪选择界面]*

#### 4.3.2 查看情绪趋势

1. 进入 **「情绪追踪」** → **「趋势查看」**
2. 选择用户与时间范围
3. 查看：
   - **情绪日历：** 以颜色标识每日整体情绪状态
   - **情绪趋势折线图：** 情绪评分随时间的变化
   - **情绪分布饼图：** 各类情绪的出现比例
   - **情绪-行为关联：** 情绪波动与行为事件的叠加分析
   - **异常预警提示：** 系统标记的异常情绪模式

> 📸 *[截图位置：情绪趋势仪表盘]*

### 4.4 沟通辅助工具使用

#### 4.4.1 使用 AAC 沟通板

1. 进入 **「沟通辅助」** 模块
2. 系统默认显示当前用户的个性化沟通板
3. 沟通板包含多个分类标签页：
   - 🏠 日常需求（吃饭、喝水、上厕所、休息等）
   - 😊 情绪感受（开心、难过、生气、害怕等）
   - 👥 社交互动（你好、谢谢、对不起、我需要帮助等）
   - 📚 学习相关（我需要帮助、我不明白、我想休息等）
   - ⚕️ 健康相关（我疼、我不舒服、我需要吃药等）
4. **使用方式：**
   - 点击图标/按钮 → 对应文字出现在底部表达栏 → 点击 **「说出」** 语音播报
   - 可连续选择多个内容构建完整句子
   - 长按按钮可编辑或删除自定义项目

> 📸 *[截图位置：AAC 沟通板主界面]*

#### 4.4.2 自定义沟通板

1. 进入 **「沟通辅助」** → **「设置」**
2. 点击 **「编辑沟通板」**
3. 可进行操作：
   - **添加新项目：** 选择图标 + 输入文字 + 选择语音
   - **调整布局：** 拖拽按钮排列位置
   - **分类管理：** 新增/编辑/删除分类标签
   - **导入模板：** 从预设模板快速创建
4. 点击 **「保存」** 生效

> 💡 **提示：** 自定义沟通板会在所有已登录设备间自动同步。

> 📸 *[截图位置：沟通板编辑界面]*

#### 4.4.3 沟通板模板库

系统提供多种预设沟通板模板，方便快速创建适合特定场景的沟通板，无需从零开始配置。

**预设模板列表：**

| 模板名称     | 适用场景            | 包含分类                      | 适合人群            |
| -------- | --------------- | ------------------------- | --------------- |
| 日常需求模板   | 家庭/日常生活         | 饮食、穿着、卫生、休息、活动            | 所有用户，尤其是初学者     |
| 学校场景模板   | 课堂/校园环境         | 课堂参与、求助、社交、课间活动、情绪表达      | 学龄期用户           |
| 医院就诊模板   | 医疗就诊/检查         | 症状描述、疼痛位置、需求表达、检查配合、情绪安抚  | 需就医的用户          |
| 情绪表达模板   | 情绪识别与表达训练       | 基础情绪、复杂情绪、情绪原因、应对策略       | 情绪识别困难的用户       |
| 社交互动模板   | 社交场景练习          | 问候、对话、邀请、拒绝、道歉            | 社交能力待提升的用户      |
| 职业训练模板   | 工作场所/职业训练机构     | 工作任务、求助、休息、同事互动、安全提示      | 参与职业训练的青少年/成人用户 |

**导入和使用模板：**

1. 进入 **「沟通辅助」** → **「设置」** → **「模板库」**
2. 浏览或搜索所需模板
3. 点击模板可预览其包含的所有按钮与分类
4. 点击 **「导入」** 将模板加载到当前用户的沟通板
5. 导入后可进一步自定义：
   - 添加/删除按钮
   - 修改按钮文字与图标
   - 调整分类与布局
6. 点击 **「保存」** 生效

> 💡 **提示：** 导入模板不会覆盖已有内容，模板内容会追加到现有沟通板中。如需替换，请先清空当前沟通板再导入。

> 📸 *[截图位置：沟通板模板库页面]*

#### 4.4.4 语音与输出设置

AAC 沟通工具的语音输出效果直接影响沟通体验，系统提供丰富的语音与输出调节选项。

**语音选择与语速调节：**

1. 进入 **「沟通辅助」** → **「设置」** → **「语音与输出」**
2. **语音选择：** 从系统提供的语音列表中选择（含男声/女声/童声选项）
3. **语速调节：** 拖动滑块调整语速（范围：0.5x - 2.0x，默认 1.0x）
4. 点击 **「试听」** 预览当前语音效果

**音量设置：**

1. 在「语音与输出」页面找到 **「音量」** 设置
2. 拖动滑块调整播报音量（0-100%）
3. 可开启 **「自动音量」**，系统会根据环境噪音自动调节

**多语言语音切换：**

1. 在「语音与输出」页面找到 **「语言」** 设置

## 5. 常见问题与故障排除

### 账户相关

**Q：忘记密码怎么办？**
A：在登录页面点击「忘记密码」，输入注册手机号或邮箱，接收验证码后设置新密码。

**Q：可以更换角色吗？**
A：可以。进入「我的」→「账户设置」→「角色信息」提交角色变更申请，机构用户需管理员审核，个人用户自动生效。

**Q：一个账号可以管理多个用户吗？**
A：可以。家属可添加多位家庭成员，教师可管理班级学生，治疗师可管理多位患者。

### 功能相关

**Q：评估做到一半退出了，数据会丢失吗？**
A：不会。评估数据自动保存，重新进入后可从上次进度继续。

**Q：AAC 沟通板可以离线使用吗？**
A：可以。已下载的沟通板内容支持离线使用，语音播报功能在首次使用时需要联网下载语音包。

**Q：为什么报告生成需要等待？**
A：系统需要分析全部相关数据并生成专业的解读文字。一般 10-60 秒完成，数据量较大时可能稍长。

### 技术相关

**Q：App 闪退或卡顿怎么办？**
A：

1. 检查网络连接是否正常
2. 尝试关闭 App 后重新打开
3. 在「设置」→「数据管理」中清除缓存
4. 检查是否有可用更新，升级到最新版本
5. 如问题持续，联系技术支持并提供设备型号与系统版本

**Q：数据同步失败怎么办？**
A：

1. 检查网络连接
2. 在「设置」→「数据管理」中点击「立即同步」
3. 如仍有问题，尝试退出登录后重新登录
4. 联系技术支持

**Q：语音播报没有声音？**
A：

1. 检查设备音量是否开启
2. 确认设备未处于静音模式
3. 进入「设置」→「无障碍」→「语音播报」确认功能已开启
4. 尝试重新下载语音包

### 隐私与安全

**Q：我的数据安全吗？**
A：所有数据采用端到端加密传输与 AES-256 加密存储。只有您授权的协作成员才能查看相关数据。我们严格遵守数据保护法规，不会将数据用于任何未授权的用途。

**Q：如何删除我的数据？**
A：进入「我的」→「设置」→「数据管理」，可删除特定数据或注销账户（永久删除所有数据）。

## 6. 无障碍使用专项指南

MindBridge Assist 致力于为所有用户提供无障碍的使用体验。本章节针对不同障碍类型提供专项使用指南，帮助用户充分发挥系统功能。

### 6.1 视障用户使用指南

**屏幕阅读器操作：**

1. 系统完整支持 iOS VoiceOver 与 Android TalkBack 屏幕阅读器
2. 所有界面元素均配有无障碍标签（Accessibility Label），确保屏幕阅读器可正确朗读
3. 常用操作手势：
   - 单指左右滑动：在元素间移动焦点
   - 双指上滑：从顶部开始朗读当前页面
   - 双击：激活当前焦点元素
4. 评估量表页面已优化朗读顺序，确保题目、选项与进度信息按逻辑顺序播报

**语音导航：**

1. 进入 **「我的」** → **「设置」** → **「无障碍」** → **「语音导航」**
2. 开启 **「语音导航模式」**
3. 开启后系统会自动播报：
   - 页面切换提示
   - 当前页面标题与功能概述
   - 操作反馈（如"已保存"、"已提交"）
   - 错误提示与修正建议
4. 可调节语音导航的播报速度与详细程度（简洁/标准/详细）

**高对比度模式：**

1. 进入 **「我的」** → **「设置」** → **「无障碍」** → **「对比度模式」**
2. 选择 **「高对比度」**
3. 高对比度模式下：
   - 文字与背景色对比度达到 WCAG AAA 标准（≥ 7:1）
   - 所有交互元素添加高可见性边框
   - 图标与按钮使用高饱和度配色
   - 禁用装饰性动画与渐变效果

### 6.4 认知障碍用户使用指南

## 8. 隐私与数据安全说明

MindBridge Assist 高度重视用户隐私与数据安全。本章节详细说明系统如何采集、存储、使用和保护您的数据。

### 8.1 数据采集说明

**采集哪些数据：**

| 数据类别     | 具体内容                              | 采集方式         |
| -------- | --------------------------------- | ------------ |
| 账户信息     | 手机号/邮箱、密码（加密存储）、昵称、角色、头像          | 用户注册时主动填写    |
| 用户档案     | 姓名、性别、出生日期、诊断信息（可选）               | 用户/家属/教师主动填写 |
| 评估数据     | 量表作答记录、评估得分、百分位、AI 解读结果           | 用户完成评估时自动采集  |
| 行为记录     | ABC 记录数据、严重程度、时间戳、备注              | 用户/教师/治疗师主动记录 |
| 情绪日志     | 情绪状态、情绪原因、语音备注、时间戳                | 用户/家属主动记录    |
| 沟通板数据    | 自定义按钮、分类配置、模板使用记录                 | 用户使用沟通工具时自动保存 |
| 设备与日志信息  | 设备型号、操作系统版本、App 版本、崩溃日志、操作日志      | 自动采集         |

**为什么采集：**

- **账户信息：** 用于身份验证与账户安全管理
- **用户档案：** 用于个性化评估适配与报告生成
- **评估数据：** 用于生成评估结果、AI 解读与进展追踪
- **行为记录：** 用于行为模式分析与干预建议
- **情绪日志：** 用于情绪趋势分析与异常预警
- **沟通板数据：** 用于跨设备同步与个性化沟通体验
- **设备与日志信息：** 用于系统稳定性维护与问题排查

**如何使用：**

- 仅用于提供、改善和个性化系统服务
- AI 分析仅在用户数据范围内进行，不会跨用户合并分析
- 未经用户明确授权，不会将数据用于其他用途

### 8.2 数据存储与保护

**加密方式：**

| 环节       | 加密方式                     | 说明                   |
| -------- | ------------------------ | -------------------- |
| 数据传输     | TLS 1.3                  | 所有网络通信均使用端到端加密       |
| 数据存储     | AES-256                  | 所有用户数据使用 AES-256 加密存储 |
| 密码存储     | bcrypt（加盐哈希）             | 密码不可逆加密，系统无法获取明文密码   |
| 本地缓存     | 设备级加密                    | 本地缓存数据受设备加密保护       |

**存储位置：**

- 主要数据存储于中国大陆境内的云服务器（符合数据本地化要求）
- 备份数据存储于异地容灾机房，同样位于中国大陆境内
- 本地缓存数据存储于用户设备，受设备安全机制保护

**访问控制：**

- 采用最小权限原则，仅授权必要人员访问用户数据
- 所有数据访问操作均记录审计日志，保留 180 天
- 运维人员访问生产数据需双人审批，且仅可访问脱敏数据
- 协作成员仅可访问用户明确授权共享的数据范围

### 8.3 数据共享规则

**与谁共享：**

| 共享对象       | 共享内容               | 共享条件             |
| ---------- | ------------------ | ---------------- |
| 协作成员       | 用户授权共享的特定数据        | 用户主动设置共享范围与期限    |
| 合作医疗机构     | 评估报告（脱敏后）          | 用户主动授权并签署知情同意书   |
| 云服务提供商     | 加密存储的数据            | 数据加密存储，服务商无法读取明文 |
| 法律法规要求     | 特定数据               | 依据法律法规或司法程序要求    |

**在什么情况下共享：**

- 用户主动授权共享（默认不共享任何数据）
- 用户使用协作功能时，按用户设定的范围共享
- 法律法规、司法程序或政府主管部门依法要求时
- 涉及儿童人身安全等紧急情况时

**如何授权：**

1. 进入 **「协作」** → **「数据共享」** 进行共享设置
2. 或进入 **「我的」** → **「设置」** → **「数据管理」** → **「数据共享」** 生成共享链接
3. 所有共享操作均需用户明确确认，系统不会自动共享任何数据
4. 用户可随时撤销共享授权

### 8.4 用户数据权利

| 权利       | 说明                           | 行使方式                                    |
| -------- | ---------------------------- | --------------------------------------- |
| 查看权      | 查看系统持有的所有个人数据                | 「我的」→「设置」→「数据管理」→「查看我的数据」              |
| 导出权      | 导出所有个人数据                     | 「我的」→「设置」→「数据管理」→「数据导出」                |
| 删除权      | 删除特定数据或全部数据                  | 「我的」→「设置」→「数据管理」→「删除数据」                |
| 更正权      | 修改不准确或不完整的个人数据               | 直接编辑对应数据或联系客服                          |
| 撤回同意权    | 撤回之前授予的数据处理同意               | 「我的」→「设置」→「隐私设置」→「撤回同意」                |

> 💡 **提示：** 行使删除权或撤回同意权可能影响部分功能的使用。注销账户后所有数据将在 30 天内永久删除。

### 8.5 数据保留期限

| 数据类型       | 保留期限              | 到期处理方式   |
| ---------- | ----------------- | -------- |
| 账户信息       | 账户存续期间 + 注销后 30 天 | 永久删除     |
| 评估数据       | 账户存续期间 + 注销后 30 天 | 永久删除     |
| 行为记录       | 账户存续期间 + 注销后 30 天 | 永久删除     |
| 情绪日志       | 账户存续期间 + 注销后 30 天 | 永久删除     |
| 沟通板配置      | 账户存续期间 + 注销后 30 天 | 永久删除     |
| 操作日志       | 180 天             | 自动删除     |
| 崩溃日志       | 90 天              | 自动删除     |
| 云端备份       | 按用户设置的备份策略保留      | 用户可手动删除  |
| 共享链接访问记录   | 共享有效期内 + 到期后 90 天 | 永久删除     |

### 8.6 儿童隐私保护特别说明

MindBridge Assist 的用户群体中包含未成年人，我们特别重视儿童隐私保护：

1. **监护人同意：** 14 周岁以下用户注册使用需获得监护人的明示同意，注册流程中需验证监护人身份
2. **最小化采集：** 对 14 周岁以下用户仅采集提供服务所必需的最少数据，不采集非必要的个人信息
3. **禁止定向推送：** 不会对儿童用户进行个性化广告推送或商业营销
4. **严格共享限制：** 儿童用户数据仅在监护人授权范围内共享，且共享操作需监护人二次确认
5. **增强加密保护：** 儿童用户数据采用额外的安全防护措施，包括更严格的访问控制与更频繁的安全审计
6. **监护人管理权：** 监护人可查看、导出、更正、删除被监护人账户中的所有数据
7. **年龄适配：** 系统功能与界面根据用户年龄自动适配，确保内容与交互方式适合儿童使用

> ⚠️ **重要提示：** 如发现任何涉及儿童数据安全的问题，请立即联系我们的专项儿童隐私保护团队：child-privacy@mindbridge-assist.com
